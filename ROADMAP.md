# Stitch Roadmap

## Current Implementation
- FastAPI backend.
- SVG to VP3 conversion.
- Raster image tracing (vtracer).
- In-browser stitch preview.
- Customization parameters (density, length, etc.).
- Basic quality metrics.

## Planned Features

### Formats
- [ ] Export to PES, DST, EXP, and JEF.
- [ ] Native AI and PDF import.
- [ ] Machine-specific optimizations.

### Logic
- [ ] Color grouping and travel run optimization.
- [ ] Material-specific presets (Denim, Silk, etc.).
- [ ] Path efficiency improvements (TSP optimization).
- [ ] Improved underlay generation for small details.

### Interface
- [ ] Improved preview (zoom/pan).
- [ ] Simple on-canvas transformations (resize, rotate).
- [ ] Batch file processing.
- [ ] Save/load user setting profiles.

### Deployment
- [ ] Native desktop wrapper (Electron/Tauri).
- [ ] Full offline functionality.
- [ ] OpenAPI/Swagger documentation.
